# 📊 POC Chart.js

A simple **Proof of Concept (POC)** project demonstrating data visualization using **Chart.js** in a React (Vite + TypeScript) application.

---

## 🚀 Live Demo

👉 https://poc-chart-js-nf4v.vercel.app/

---

## 📌 Features

* 📈 Interactive charts using Chart.js
* ⚡ Fast build with Vite + TypeScript
* 🎯 Clean and minimal UI
* 🔄 Dynamic data rendering
* 📱 Responsive design

---

## 🛠️ Tech Stack

* React 18
* TypeScript
* Vite
* Chart.js
* CSS

---

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/your-username/POC_Chart.js.git
cd POC_Chart.js
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

---

## 🏗️ Build

```bash
npm run build
```

---

## 🚀 Deployment

Deployed using **Vercel**

* Build Command: `npm run build`
* Output Directory: `dist`

---

## ⚠️ Note

If dependency issues occur:

```bash
npm install --legacy-peer-deps
```
